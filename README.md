## 技术参数

- **输入电压**：550V
- **输出负载线电压（有效值）**：220V
- **频率**：50Hz
- **相位关系**：三相对称，各相位间相差120°
- **开关电流**：约45A

## 设计特点

- **双闭环控制**：采用电流内环和电压外环的设计结构，确保了系统具有快速的动态响应和良好的稳态性能。
- **电压调节精确**：通过外部反馈，能够精细调整输出电压，适应不同负载需求。
- **频率稳定性**：固定的50Hz输出，适合交流电源应用，保证了负载设备的工作一致性和可靠性。
- **负载适应性强**：设计考虑到了广泛的负载变化范围，能够保持输出质量的一致性。
